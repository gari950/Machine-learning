Objective/ Aim:  to evaluate numeric data dissimilarity using Minkowski distance.

Data set URL: https://www.kaggle.com/hugoncosta/price-of-flats-in-moscow

Tools: Pyhton

Theory: Minkowski distance is a metric in a normed vector space. Minkowski distance is used for distance similarity of vector. Given two or more vectors, 
find distance similarity of these vectors.

Mainly, Minkowski distance is applied in machine learning to find out distance similarity.
